# Building an Autonomous Ground Vehicle

For this class project, I focused on combining hardware engineering and software logic to build a fully functional vehicle.

## Technical Details
- **Hardware Design:** Used CAD to model the chassis and 3D-printed a custom purple outer shell.
- **Programming:** Wired the internal components and wrote the control logic for the UAV using Arduino.
- **Languages:** Utilized Python and Arduino programming to ensure smooth autonomous movement.
